# Coffee Sales Data Analysis

## Project Overview

This project analyzes coffee shop sales data using Python.

The objective is to understand product sales performance and identify top selling products.

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Analysis Performed

* Data cleaning using Pandas
* Sales analysis by product
* Revenue analysis
* Bar chart visualization of product sales

## Insights

* Identified top selling coffee products
* Compared product revenue
* Visualized sales trends using bar charts
